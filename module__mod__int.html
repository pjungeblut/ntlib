Detailed Description

Represents an element of a residue class modulo a given number.

Let \(m \in \mathbb{N}\). This class represents an element of the residue class \(\mathbb{Z}/m\mathbb{Z}\).

There are two classes in this module.

  • rt_mod_int takes the modulus \(m\) at run-time.
  • ct_mod_int takes the modulus \(m\) at compile-time.
