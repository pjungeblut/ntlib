SieveType ntlib::prime_sieve ( T N)
export

Generates a prime sieve.

Template Parameters
TThe integral data type to use.
AllocatorThe allocator to use for the list of primes.
SieveTypeThe datastructure to use as a prime sieve. NTLib provides a classical array like type and a 235-compressed version. The latter one can save half the time to generate the primes in a tradeoff for slower access times.
SEGMENT_SIZEPortion of the sieve to be processed at once. This should be small enough that SEGMENT_SIZE sieve fields fit easily into the L3 cache.
Parameters
NGenerate a prime sieve with all values up to N.
Returns
The sieve.
