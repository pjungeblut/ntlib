Detailed Description

template<typename T, typename DERIVED>
class ntlib::mod_int< T, DERIVED >

Base class for CRTP design of rt_mod_int and ct_mod_int.

Note
As this base class does not store the modulus \(n\), it makes no sense to create instances of it. For this reason, all constructors and assignment operators are protected.
